Only use this if your artifacts have a keyword tagged on them.
For example, if your artifacts are laid out like this:

reponame-dev-1.0.0.zip
reponame-dev-1.0.1.zip
reponame-main-1.0.0.zip
reponame-main-1.0.1.zip

setting this to "main" would return reponame-main-1.0.1.zip
setting this to "dev" would return reponame-dev-1.0.1.zip

Leaving this empty will default to the latest artifact available (reponame-main-1.0.1.zip)
